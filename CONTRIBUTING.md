# Contributing

Daxie is public source for transparency. The code, security model, and tradeoffs
are available for inspection because Daxie handles agent-controlled wallet
behavior and should be auditable.

Development is currently maintainer-led. At this stage, unsolicited pull
requests, feature work, and roadmap contributions are not being accepted.

Public issue intake may open more broadly after the v1 surface is settled. When
that happens, issues will be used for bug reports, questions, and narrowly scoped
feedback rather than general roadmap direction.

Security reports are handled separately through [SECURITY.md](SECURITY.md). Do
not put exploit details, wallet material, or live-funds reproductions in public
issues.

You are welcome to read the code, fork it, experiment with it, and build from it
under the project license. Pull requests should only be opened after explicit
maintainer agreement.
